Welcome to my GitHub Site

This page is intended to do ESP32 firmware update by Over-The-Air (OTA).

Firmware Update Files on Public Storage
  • firmware_x.bin — 'x' indicates version number, e.g. firmware_1.0.bin, firmware_1.1.bin, firmware_2.7.bin, etc.
  • firmware.json — informs the ESP32 that a firmware update is available.
